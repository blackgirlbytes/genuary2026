This repo contains two experimental approaches for creating Genuary art with Goose:

| Approach | Directory | How it works |
|----------|-----------|--------------|
| **Harness** | `genuary/` | Recipe + shell script for scaffolding |

- Find the daily prompt
- Pitch whimsical, creative ideas
- Write p5.js code
- Verify with Chrome DevTools
- Save GIF/video + PNG outputs
- Push to GitHub
